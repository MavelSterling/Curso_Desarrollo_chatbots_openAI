# Curso de Desarrollo de Chatbots con OpenAI

Este repositorio contiene materiales y ejemplos prácticos para un curso sobre el desarrollo de chatbots utilizando la API de OpenAI de Platzi. El curso está diseñado para principiantes y desarrolladores intermedios que deseen aprender a integrar modelos de lenguaje de OpenAI en aplicaciones de chatbots.

## Contenido del Curso

### 1. Introducción a Chat Completions

- **Archivo**: `chat_completion.ipynb`
- **Descripción**: Ejemplo básico de cómo usar la API de OpenAI para generar respuestas de chat. Incluye instalación de la librería, configuración de la API key y un ejemplo práctico de conversación.

### 2. Clasificación de Tweets

- **Archivos**: `tweets_classifier_original.ipynb`, `Plantilla_tweets_classifier_original.ipynb`
- **Descripción**: Tutoriales para crear un clasificador de tweets utilizando OpenAI. Aprende a analizar sentimientos, categorizar contenido y automatizar la clasificación de texto en redes sociales.

## Requisitos

- Python 3.7 o superior
- Una cuenta de OpenAI con API key válida
- Librerías: `openai` (instalable via pip)

## Instalación

1. Clona este repositorio:

   ```bash
   git clone https://github.com/MavelSterling/Curso_Desarrollo_chatbots_openAI.git
   cd Curso_Desarrollo_chatbots_openAI
   ```
2. Instala las dependencias:

   ```bash
   pip install openai
   ```
3. Configura tu API key de OpenAI en los notebooks (reemplaza `'Ingresa aquí tu API Key de OpenAI'` con tu clave real).

## Uso

1. Abre los notebooks en Jupyter Notebook o JupyterLab.
2. Ejecuta las celdas en orden.
3. Experimenta modificando los prompts y parámetros para ver diferentes resultados.

## Estructura del Repositorio

- `chat_completion.ipynb`: Notebook de introducción a chat completions.
- `tweets_classifier_original.ipynb`: Implementación completa del clasificador de tweets.
- `Plantilla_tweets_classifier_original.ipynb`: Plantilla base para proyectos de clasificación.
- `README.md`: Este archivo.
